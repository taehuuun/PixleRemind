# 픽셀 리마인드 (PixelRemind)

색을 잃어버린 픽셀 세상의 원래의 색을 복원 하는 퍼즐 형식의 게임 입니다.

## 개발 환경
- [Unity Game Engine](https://unity3d.com) (version: 2021.3.16f1)
- C#
- Firebase
